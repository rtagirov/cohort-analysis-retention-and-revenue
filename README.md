# Cohort Analysis: Retention & LTV (Online Retail)

Kогортный анализ удержания клиентов и формирования LTV на основе реальных транзакционных данных онлайн-ритейла.

## Описание проекта

В рамках проекта выполнен анализ клиентского поведения во времени:
- как клиенты возвращаются после первой покупки,
- как формируется выручка по когортам,
- за счёт каких клиентов и периодов создаётся основной LTV.

Анализ ориентирован на бизнес-интерпретацию, а не только на визуализацию метрик.

## Датасет

Используется датасет Online Retail II — публичный набор данных с транзакциями британской B2B-компании, занимающейся оптовой онлайн-торговлей.

Характеристики данных:
- период: декабрь 2009 — декабрь 2011,
- тип бизнеса: B2B e-commerce,
- данные о заказах, клиентах, количестве и стоимости товаров,
- выраженная повторяемость покупок и длинный жизненный цикл клиентов.

## Что сделано в проекте

- Очистка и подготовка транзакционных данных
- Формирование когорт по месяцу первой покупки
- Расчёт:
  - retention по когортам,
  - выручки по месяцам жизни клиента,
  - накопленного LTV (revenue-based)
- Визуализация:
  - heatmap удержания,
  - heatmap LTV по когортам
- Анализ структуры выручки внутри когорт и влияния крупных клиентов

## Ключевые выводы

- Основная часть LTV формируется в первые месяцы жизни клиента, однако значимый вклад продолжают вносить повторные покупки отдельных клиентов.
- В ранних когортах наблюдается высокая концентрация выручки у небольшого числа клиентов, что характерно для B2B-модели.
- Различия между когортами объясняются не только удержанием, но и структурой заказов и частотой покупок.
